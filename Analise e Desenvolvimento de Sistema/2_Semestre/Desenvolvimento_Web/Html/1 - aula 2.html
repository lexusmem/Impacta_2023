TAG'S DE ESTRUTURA

TAG DIV

Este paragrafo esta contido em uma div que é uma tag de divisão da página. A tag div, também conhecida como "divisão", é um elemento fundamental na construção de layouts em HTML. Apesar de ser genérica, ela possui diversas funções importantes:

  • Agrupar Conteúdo:
    Seu principal objetivo é agrupar elementos HTML, como parágrafos, imagens, títulos e outros blocos. Isso facilita a organização do conteúdo e a aplicação de estilos de forma modular. Imagine um site com seções distintas, como cabeçalho, menu lateral, conteúdo principal e rodapé. Cada seção pode ser criada usando uma tag div.
  • Estilização com CSS:
    A tag div em si não possui estilos visuais pré-definidos. Mas, combinando-a com CSS, você pode adicionar formatação e layout personalizados aos elementos agrupados. Através de classes e IDs, é possível definir cores, fontes, espaçamentos, posicionamento e diversos outros aspectos visuais. Isso torna a construção de layouts mais flexível e eficiente.
  • Semântica com Atributos:
    Embora não seja semântica por padrão, a tag div pode ser enriquecida com atributos para fornecer mais informações sobre o conteúdo agrupado. Por exemplo, o atributo id permite criar um identificador único para a seção, enquanto o atributo class pode ser usado para categorizar ou agrupar diversas divs com características semelhantes.
  • Flexibilidade e Acessibilidade:
    A simplicidade da tag div a torna extremamente versátil. Você pode usá-la para estruturar diversos tipos de conteúdo, desde layouts complexos até seções simples. Além disso, sua natureza semântica, quando utilizada com atributos adequados, contribui para a acessibilidade do site, pois facilita a compreensão da estrutura da página para leitores de tela e outros dispositivos assistivos.

TAG SECTION

Outra tag de estrutura é a section.

A tag section em HTML é usada para definir uma seção genérica em um documento. Pense nela como uma forma de agrupar conteúdos relacionados por tema, como capítulos em um livro, artigos em um jornal ou diferentes partes de um formulário.

  • Semântica: Ao invés de ser apenas uma tag genérica de layout, section é um elemento semântico, o que significa que ele descreve o significado do conteúdo para navegadores e desenvolvedores. Isso ajuda na acessibilidade e facilita a compreensão da estrutura da página.
  • Uso geral: Se não houver uma tag mais específica para o seu tipo de conteúdo, a section é uma boa opção. Por exemplo, você pode usá-la para agrupar comentários em um blog ou produtos em uma página de loja virtual.
  • Títulos: Embora não seja obrigatório, é recomendável incluir um título (h1 até h6) dentro da seção para identificar seu conteúdo.

TAG ARTICLE

Outra tag estrutura de article.

A tag article serve para marcar um conteúdo autossuficiente dentro de uma página, aplicação ou site. Imagine um artigo de jornal, um post de blog, um produto individual em uma loja online - cada um desses conteúdos é independente e poderia, em teoria, ser lido ou reutilizado separadamente. É para este tipo de conteúdo que usamos a tag article.

A ideia por trás do article é que o conteúdo possa ser distribuído ou reutilizado de forma independente, por exemplo, em feeds RSS.
